## ✦ What is Notchturne?

Notchturne brings the **macOS Dynamic Island** experience to Windows — sitting silently at the top of your screen as a pure-black notch, and coming alive when something worth showing happens.

No taskbar clutter. No widgets. Just a sleek, intelligent bar that shows you exactly what matters — right now.

---

## ✦ Features

| Feature | Description |
|---|---|
| 🎵 **Now Playing** | Live music display with track, artist & playback state |
| 🔋 **Battery Awareness** | Charging and discharging status with animated indicator |
| 🔊 **Volume Control** | System volume at a glance, right in the notch |
| 🤖 **Claude Usage Tracker** | Monitor your Anthropic Claude API usage in real time |

## ✦ Installation

### Requirements

- Windows 10 / 11
- .NET 8 Runtime *(or whichever you're using — adjust accordingly)*

### Quick Start

```bash
# Download the latest release
# → Releases tab on the right →

# Or clone and build yourself:
git clone https://github.com/Lippi304/notchturne.git
cd notchturne
```

> **Note:** A standalone `.exe` installer is planned for the first stable release.
